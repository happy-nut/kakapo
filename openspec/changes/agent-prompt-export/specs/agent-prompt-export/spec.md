## ADDED Requirements

### Requirement: 리뷰를 에이전트 프롬프트 파일로 내보내기
사용자는 저장된 리뷰 코멘트 전체(질문 + 수정요청)를 에이전트가 읽을 수 있는 마크다운 프롬프트 파일로 내보낼 수 있어야 한다(SHALL). 파일은 자기완결적이어야 한다(MUST) — 각 항목은 `file:line`, 캡처된 코드(있으면), 코멘트 텍스트를 포함한다.

#### Scenario: 합본 뷰에서 프롬프트 파일 저장
- **WHEN** 리뷰 코멘트가 하나 이상 있고 사용자가 합본 뷰에서 "Save prompt"(또는 해당 단축키)를 실행할 때
- **THEN** 질문과 수정요청을 모두 담은 마크다운 프롬프트가 `.monacori/review-prompt.md`에 기록되고, 저장된 경로가 사용자에게 표시된다

#### Scenario: 프롬프트가 자기완결적이다
- **WHEN** 내보낸 프롬프트 파일을 에이전트가 읽을 때
- **THEN** 각 코멘트의 파일·줄·캡처된 코드·요청 텍스트가 포함되어, 원본 diff 없이도 에이전트가 무엇을 고칠지 이해할 수 있다

#### Scenario: 파일 쓰기가 불가능한 환경의 폴백
- **WHEN** 파일 쓰기가 불가능한 환경(정적 브라우저 빌드, IPC 부재)일 때
- **THEN** 기존 "Copy all" 복사 동작으로 폴백하여 사용자가 프롬프트를 클립보드로 가져갈 수 있다
